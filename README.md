**Online Food Delivery App**
QuickBite üçî is a Flutter-based online food delivery application developed as a semester project. The app allows users to browse food categories, view restaurants, explore menus, add items to a cart, and place orders via WhatsApp using a guest checkout system.

**Features**

1.Splash screen with app branding 
2.Guest access (no login or signup required) 
3.Category-based food browsing 
4.Restaurant listing per category 
5.Menu with item images and prices 
6.Add to cart functionality 
7.Cart summary with total calculation 
8.Order confirmation via WhatsApp 
9.Clean and responsive UI using Material Design

Tech Stack Flutter Dart Material UI URL Launcher (WhatsApp integration)
App Flow Splash Screen Guest Entry Food Categories Restaurant List Menu Items Cart Summary Order Confirmation via WhatsApp

Project Purpose This project was built for learning and academic purposes to understand:
Flutter navigation State management basics UI design principles Data handling using maps and lists Real-world app flow simulation
